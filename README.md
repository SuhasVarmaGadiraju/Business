# Pickles

Static website for **Godavari Homemade Pickles**.

## Live Demo

- Deploy on Vercel and paste your URL here.

## Run Locally

- Open `index.html` in a browser.

## Deploy on Vercel

- Import this GitHub repo into Vercel.
- Framework preset: **Other** (no build step).
- Vercel will serve `index.html` at `/` using `vercel.json`.

## Project Structure

- `index.html`
- `order.html`
- `summary.html`
- `static/`
  - `app.css`
  - `main.js`
  - `images/`

## Notes

- All navigation uses plain HTML links.
- Assets use relative paths like `./static/app.css` and `./static/main.js`.
